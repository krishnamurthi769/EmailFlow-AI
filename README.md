# EmailFlow AI

A smart email drafting assistant powered by **Google Gemini 2.0 Flash**. This tool helps professionals generate structured, tone-specific emails instantly using simple bullet points.

## Features
* **Context-Aware Drafting:** Converts shorthand notes into full professional emails.
* **Tone Selection:** Choose between Professional, Friendly, Formal, or Concise tones.
* **Secure API Handling:** Uses `.env` to manage sensitive API keys securely.
* **Smart Highlights:** Automatically highlights key terms (attachments, dates, requests) in the preview.
* **Copy-Paste Ready:** Provides a raw text view for easy copying.
